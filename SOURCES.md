# Sources and clean-room statement

This repository is a clean-room implementation written from functional requirements and first-hand experience building evidence-backed knowledge systems.

No third-party Skill prose, scripts, test fixtures, hidden markers, or proprietary expert corpus are incorporated.

- OpenAI, “Build plugins”: https://learn.chatgpt.com/docs/build-plugins.md

If future versions incorporate licensed third-party code or substantial text, record the source, license, files, and modifications in this document before release.
